# Рекомендательная система на основе логических правил

Данный репозиторий содержит программный код и результаты экспериментов для выпускной квалификационной работы на тему «Применение математического моделирования и искусственного интеллекта для разработки рекомендательных систем».

## Структура репозитория

### Основные модули
- `system.py` — реализация рекомендательной системы (класс `RecommenderSystem` с гипотезами H1–H6)

### Экспериментальные скрипты
- `experiment_task2.py` — апробация системы на датасете Yandex Yambda-50M (оценка покрытия)
- `experiment_task3.py` — анализ динамики популярности треков, проверка модели «хищник–жертва» (Лотки–Вольтерры)
- `experiment_task4.py` — сравнение различных конфигураций системы

### Презентационные скрипты
- `presentation_demo_advanced.py` — демонстрация всех комбинаций гипотез для пользователей из ВКР

### Результаты и данные
- `processed_data_task4.pkl` — кэш обработанных данных (лайки, дизлайки, сильные прослушивания)
- `event_timeline_data.pkl` — кэш временных рядов для модели «хищник–жертва»
- `predator_prey_pairs_table.csv` — таблица пар «хищник–жертва» с корреляциями
- `comparison_results.csv` — результаты сравнения конфигураций системы

### Графики и визуализация
- `figure_3_1a.png`, `figure_3_1b.png` — динамика популярности топ-10 треков
- `figure_3_2.png` — график пары «хищник–жертва»
- `figure_3_3_predator_prey_pair.png` — дополнительный график пары
- `popularity_dynamics.png`, `lotka_volterra_model.png` — иллюстрации модели
- `predator_prey_pairs/` — папка с графиками всех найденных пар

## Краткое описание работы

Разработана логическая рекомендательная система на основе продукционных правил «если-то» (6 гипотез H1–H6). Система протестирована на датасете Yandex Yambda-50M при разреженности матрицы 99,995%. Покрытие составило 96,7% (967 из 1000 пользователей).

Также проведён анализ динамики популярности музыкальных треков с применением модели Лотки–Вольтерры («хищник–жертва»). Выявлены 30 треков-«хищников», 39 треков-«жертв» и 2648 треков с осцилляциями. Максимальная отрицательная корреляция –0,51.
